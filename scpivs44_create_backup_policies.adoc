= バックアップポリシーの作成
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
バックアップポリシーを作成するには、次の手順に従います。

. SCVプラグインの左側のナビゲータペインで、*[Policies]*をクリックし、*[New Policy]*を選択します。
. [ 新しいバックアップ・ポリシー * ] ページで、ポリシー設定情報を入力し、 [ * 追加 ] をクリックします。
